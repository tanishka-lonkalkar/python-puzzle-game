A fun interactive web-based learning game that combines Python quiz questions with a color-matching drag puzzle.
Designed as an educational mini-project to make learning Python more engaging and game-like.
Features
🧠 Python quiz system (Easy, Medium, Hard levels)
🎯 10+ questions per level (expandable)
🎨 Attractive dark-themed UI
🟢🟡🔵🔴 Color matching puzzle game
🖱️ Drag & connect puzzle interaction
📊 Score tracking system
❌ "Try Again" for wrong answers
🎉 Puzzle unlock after correct answer
🔁 Next question system
📱 Mouse + touch support
Game Flow
Select difficulty level (Easy / Medium / Hard)
Answer Python MCQ question
If correct → Puzzle appears
Solve color matching puzzle
Click “Next Question”
Repeat until level ends
Final score is displayed
🧩 Puzzle Concept
A 4x4 grid of colored dots
Each color appears in pairs
User must connect matching colors using drag
Puzzle completes when all pairs are connected
